# Unbeatable Football Website

React landing page for Unbeatable Football, a premium historical football draft game.

## Local Development

### Prerequisites
- Node.js v16 or higher
- Yarn package manager

### Setup
1. Install dependencies:
   ```bash
   yarn install
   ```

2. Start the development server:
   ```bash
   yarn start
   ```

The application will be available at `http://localhost:3000`.

## Deployment on Vercel

1. Connect the repository to Vercel.
2. Vercel uses the configuration in `vercel.json`.
3. No environment variables are required for the static landing page.

## Project Structure
- `frontend/` - React application using Create React App with CRACO
- `vercel.json` - Vercel deployment configuration
- `package.json` - Root package configuration with workspaces

## Build

```bash
yarn build
```

## Product Positioning

Unbeatable Football lets fans draft players from legendary football team-seasons across 11 rounds, build a lineup, clear rating or opponent challenges, and protect daily challenge streaks across clubs, countries, tournaments, and eras.
